For this project, I was given the core environment code, including the TreasureMaze class that defines the maze structure, movement rules, and reward system, along with the experience replay framework. That code handled how the pirate moves and how the game state is represented. What I built myself was the deep Q learning training loop, including the qtrain function, the epsilon greedy strategy for exploration and exploitation, and the neural network model used to approximate Q values. I connected the environment to the learning algorithm and tuned parameters so the agent could improve over time and consistently find the treasure.

This project reflects what computer scientists do in general. We design algorithms and systems to solve problems efficiently using abstraction and logical reasoning. Instead of hard coding a path, I approached the maze as a state space problem and applied reinforcement learning to let the agent learn from interaction. That mindset of breaking problems into models, selecting appropriate algorithms, and testing performance is central to computer science and is what allows us to solve complex real world challenges.
